Политика приватности — TreeLapse

Дата вступления в силу: 30 июня 2026 г.

TreeLapse — приложение для фотодневника деревьев и растений. Эта политика рассказывает, какие данные собирает приложение, где они хранятся и кому могут передаваться.

Коротко: почти все данные — фото деревьев, GPS-координаты, настройки — хранятся только в памяти вашего устройства. На сторонний сервер уходит лишь одно фото при первом распознавании вида растения через PlantNet (см. ниже).

Какие данные собирает приложение

  • Фото деревьев, которые вы делаете в приложении. Хранятся локально в защищённой папке приложения и не передаются никуда без вашего явного действия (кроме случая распознавания вида — см. ниже).
  • GPS-координаты точки, где сделан первый снимок дерева (только если вы разрешили доступ к локации). Используются исключительно для отображения дерева на карте внутри приложения. Координаты никуда не передаются.
  • Метаданные дерева: ваше имя для дерева, дата создания, результат распознавания вида (если он был получен). Хранятся локально.
  • Настройки приложения: единицы измерения, выбранная скорость таймлапса, параметры стабилизации. Хранятся локально.

Распознавание вида растения (PlantNet)

При создании нового дерева приложение по умолчанию отправляет одно фото на сервис PlantNet для определения вида растения. В ответ приложение получает название вида, семейство и оценку уверенности.

PlantNet — независимый научный сервис (Cirad, INRA, INRIA, IRD — Франция), используемый в исследовательских целях по биоразнообразию. Их политика приватности доступна на plantnet.org.

На сервер уходит только сама фотография — ни GPS-координаты, ни ваше имя дерева, ни любые другие идентификаторы. По завершении распознавания результат сохраняется локально, повторных запросов для того же дерева не делается. Кнопка «Распознать заново» в карточке дерева отправляет фото повторно — только по вашему явному действию.

Карта (OpenStreetMap)

Экран карты использует тайлы (фрагменты карты) сервиса OpenStreetMap. При просмотре карты ваш телефон запрашивает у их серверов фрагменты изображений — это стандартное поведение для любого приложения, отображающего карты. OpenStreetMap не получает информацию о ваших деревьях или GPS-точках, отмеченных в TreeLapse.

Аналитика и реклама

Приложение не использует сторонних трекеров, аналитики поведения или рекламных сетей. Ни Google Analytics, ни Firebase Analytics, ни AppMetrica, ни любых других подобных сервисов.

Где хранятся данные

Все данные хранятся только в памяти вашего устройства, в защищённой папке приложения, доступной исключительно TreeLapse. Резервные копии в облако (Google Drive, iCloud) автоматически не делаются. Если вы переустановите приложение или сбросите телефон — данные будут утеряны.

Удаление данных

Чтобы удалить все данные о конкретном дереве (включая фото и собранные таймлапсы), нажмите «Удалить дерево» в меню карточки. Чтобы удалить все данные приложения сразу — удалите TreeLapse через стандартные настройки Android.

Поддержка проекта (Boosty)

В разделе «Настройки → Поддержать» есть ссылка на страницу автора на платформе Boosty для добровольных донатов. Переход и оплата происходят полностью внутри Boosty по их правилам и политике. Приложение TreeLapse не получает никакой информации о ваших платежах.

Возраст пользователей

Приложение не предназначено специально для детей младше 13 лет. Сознательного сбора данных детей не ведётся.

Изменения политики

Политика может обновляться по мере добавления новых функций. Существенные изменения будут отмечены датой вступления в силу в начале документа.
